# 🤖 Assistente de Estudos com IA — Gemma 2 (Groq API + Streamlit)

Este projeto é um **assistente inteligente de estudos** focado em explicar conteúdos de programação de forma simples, clara e intuitiva.  
Ele utiliza o modelo **Gemma 2 - 9B Instruct**, executado pela **Groq API**, conhecida por oferecer **inferência ultrarrápida e gratuita**.

Disponibiliza duas interfaces:

- 🖥 **CLI (modo terminal)**
- 🌐 **Interface Web usando Streamlit**

## 📚 Funcionalidades Principais

- Explicação de conceitos de programação (POO, SQL, Java, Python etc.)
- Respostas didáticas, diretas e com exemplos
- Interface gráfica acessível via navegador
- Uso seguro de API Key com Secrets do Streamlit
- Integração real com LLM via Groq API
- Baixíssimo custo e alta performance
